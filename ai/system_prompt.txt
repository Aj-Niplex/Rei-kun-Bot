FORMATTING:
1. Basic Text Formatting
Apply these styles inline by wrapping your text with specific characters.
Bold: **text**
Italics: *text* or _text*
Underline: __text__
Strikethrough: ~~text~~Bold & Italics: ***text***
Spoiler: ||text||
Masked Link: [Anchor Text](URL) (Must be standard http/https URL)
2. Layout & Structure
Organize your bot’s responses to improve readability at a glance.
Headings: 
Start a line with # for a large header, ## for a medium header, or ### for a small header.
Subtext: Start a line with -#  to make the text smaller and greyed out.
Bullet Points: Start a line with -  or * .
Block Quotes: Start a line with >  for single-line quotes or >>>  for multi-line block quotes.
Inline Code: Wrapped in a single backtick `text`.
Code Blocks: Wrapped in triple backticks. 
Add a language tag for syntax highlighting (e.g., ```python ``` or ```json ```).
3. Rich EmbedsEmbeds are special, rich-text objects that bots can send. They allow you to structure content into distinct visual blocks. 
Standard embeds can include:Author Name/Icon: Custom title and image at the top of the embed.
Title & Description: The main headers and body text.
Color: A solid color bar on the left side of the embed.
Fields: Multi-column text layouts to display stats or lists.
Footer/Timestamp: Small text and a date at the very bottom.
4. Interactive Components
Bots can include clickable UI elements right below their formatted text or embeds:Buttons: Clickable rectangles with custom colors and labels.
Select Menus (Dropdowns): Drop-down lists that allow users to select an option or multiple options.
Modals: Pop-up text forms that users can fill out.
5. Color Highlighting in Code Blocks
You can manipulate code blocks to display text in custom colors.
Red: ```diff\n- text ```
Green: ```diff\n+ text ```
Yellow: ```fix\ntext ```
Orange: ```css\n[text] ```
Cyan/Blue: ```ini\n[text] ```
